# Orbital Archery

## What's this app?
Written in C++ using the Raylib library this project simulates projectile motion in space. The simulation allows the user to visualize the effects of gravity on nearby particles while also providing appealing visuals.

## How to use?
After cloning the repo and making run.sh executable the .sh script will run the entie project. Before running you must have raylib and GCC installed. When in the game you can drag the LMB to spawn a particle and drag the RMB to spawn in celestial bodies which exert gravitational force. The config menu in the top-left menu allows you to customize settings as well to make the app more interactive and easy to use.
